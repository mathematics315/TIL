# コメント修正方法

## 直前のコミット

```
$ git commit --amend -m "修正メッセージ"
```

## 2つ以上前のコミット

- 一番新しいコミットを1として、3番目のコミットメッセージを修正したい場合

  ```
  $ git rebase -i HEAD~3
  
  ```
  
  エディタが起動するので、上の方にある修正したいコミットの「 __pick__ 」というところを、「 __edit__ 」に変更して保存する
  エディタを閉じる
  
  ```
  $ git commit --amend -m "修正メッセージ"
  $ git rebase --continue
  ```
